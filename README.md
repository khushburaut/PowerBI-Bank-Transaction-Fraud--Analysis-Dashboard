# Bank Transaction Fraud Detection Dashboard

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze bank transaction data and identify fraudulent activities. The dashboard provides visual insights into transaction patterns, fraud distribution, payment methods, device usage, and customer transaction behavior.

The goal of this project is to help detect suspicious transactions and support data-driven fraud prevention strategies.

## Objectives

- Analyze transaction patterns across different categories
- Identify fraudulent and non-fraudulent transactions
- Monitor fraud occurrences and trends
- Understand payment behavior and device usage
- Support decision-making using visual analytics

## Tools & Technologies Used

- Power BI
- Microsoft Excel / CSV Dataset
- DAX (Data Analysis Expressions)
- Power Query

## Dashboard Features

- KPI cards displaying:
  - Average Transaction Amount
  - Total Transactions
  - Total Fraud Cases

- Interactive visualizations:
  - Fraud vs Non-Fraud Transactions
  - Transaction Count by Country
  - Payment Method Analysis
  - Merchant Category Analysis
  - Device Type Analysis
  - Fraud Case Distribution

- Interactive filtering and drill-down capabilities

## Key Insights

- Total transactions analyzed: **1M**
- Total fraud cases identified: **55K**
- Fraudulent transactions represent a smaller portion of total transactions
- Credit Card and Debit Card transactions are among the most commonly used payment methods
- Fraud patterns can be analyzed across countries, merchant categories, and device types
